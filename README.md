# Deezer Application

[Demo available here](https://deezer-application.netlify.app)

## Getting Started

Install dependencies :

```bash
yarn install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
